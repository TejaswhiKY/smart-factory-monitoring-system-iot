
## 🎥 Demo Video

This video demonstrates the working of the **Smart Factory Monitoring System using IoT**.


In this demo:


- Node 1 (Production Line) monitors machine parameters like temperature and vibration  
- Node 2 (Warehouse) monitors environmental conditions like light, humidity, and motion  
- Real-time data is sent using Arduino and W5500 Ethernet  
- Data is visualized on the ThingsBoard dashboard  
- LED indicators and alerts show system status
